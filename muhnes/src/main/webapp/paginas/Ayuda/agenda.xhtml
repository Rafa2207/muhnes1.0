Permite realizar diferentes acciones sobre el control de los proyecto, entre estas estan: visualizar el porcentaje de desarrollo de los proyectos, dirigirse a la agenda donde se controlarán las actividades y proyectos, si el proyecto se encuentra activo podrá desactivarlo en caso contrario tendrá la opción de activarlo.

Agenda:
Permite controlar, visualizar y agregar prórrogas a las actividades del proyecto seleccionado.

Desactivar:
Deshabilita el proyecto seleccionado por el usuario.

Activar:
Si el proyecto se encuentra desactivado se habilitara esta opción, para poder reestablecer el estado en el que originalmente permanecio antes de desactivar dicho proyecto. En caso contrario esta acción permanecera deshabilitada.

Pasos para dirigirse a la agenda.

Paso 1:
Seleccióne el proyecto.

Paso 2:
Se habilitara el botón Agenda para luego hacer clic y dirigirse a la agenda.

Pasos para desactivar el proyecto.

Paso 1:
Seleccióne el proyecto que desea desactivar.

Paso 2:
Se habilitara el botón Desactivar, luego hacer clic y se presentara el formulario. En caso contrario el botón aparecera deshabilitado.

Paso 2:
Ingresa los datos que debe digitar en el formulario, luego hacer clic en botón Guardar.

Paso 3:
Si desea cerrar el formulario, haga clic en el botón X o Cerrar.

Pasos para activar el proyecto.

Paso 1:
Seleccióne el proyecto que desea activar.

Paso 2:
Se habilitara el botón Activar, luego hacer clic y se presentara el formulario. En caso contrario el botón aparecera deshabilitado.

Paso 2:
Ingresa los datos que debe digitar en el formulario, luego hacer clic en botón Guardar.

Paso 3:
Si desea cerrar el formulario, haga clic en el botón X o Cerrar.
