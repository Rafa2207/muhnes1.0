Permite realizar diferentes acciones en los perfiles como registrar, visualizar y modificar, así como tambien podrá generar reportes de manera general.

1. Nuevo:
Registra un nuevo perfil.

2. Visualizar:
Muestra el contenido del perfil.

3. Modificar:
Actualiza los datos ingresados anteriormente del perfil seleccionado.

Nombre:
Ingrese el nombre del perfil.

Descripción:
Ingrese una breve descripción del nuevo perfil.

Guardar:
Almacena a la base de datos la información ingresada en el formulario.

(X)Cerrar:
Cierra el formulario sin almacenar los datos ingresados.

(*) Campo requerido:
Al aparecer este símbolo en alguna etiqueta quiere dar a entender que el campo es requerido u obligatorio. En otras palabras si no es ingresado el dato solicitado en el campo, no se podrá almacenar en la base de datos del sistema.

Pasos para visualizar el perfil.

Paso 1:
Seleccióne el perfil que desea visualizar.

Paso 2:
Se habilitara el botón Visualizar para luego hacer clic y ver la información del perfil seleccionado.

Paso 3:
Si desea cerrar el formulario, haga clic en el botón X o Cerrar.

Pasos para modificar los datos del género seleccionado.

Paso 1:
Seleccióne el perfil que desea modificar o actualizar la información.

Paso 2:
Se habilitara el botón Modificar, luego hacer clic y el formulario mostrara los datos del perfil para que puedan ser editados.

Paso 3:
Realize los cambios que desea y haga clic en el botón Guardar.

Paso 4:
Si desea cerrar el formulario, haga clic en el botón X.
