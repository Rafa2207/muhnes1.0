Permite realizar diferentes acciones sobre los proyectos como por ejemplo: registrar, visualizar, modificar o generar reportes de manera individual o general, así como también administrar actividades o prórrogas del proyecto seleccionado.

Nuevo:
Registra un nuevo proyecto.

Visualizar:
Muestra el contenido del proyecto seleccionado, además permite generar reporte planificado y final.

Modificar:
Edita los datos ingresados anteriormente en el proyecto seleccionado.

Actividad:
Permite gestionar(registrar, visualizar y modificar) las actividades del proyecto seleccionado.

Prórrogas:
Ofrece al usuario la gestión sobre las prórrogas del proyecto seleccionado,en esta opción el usuario podrá registrar, visualizar y modificar prórrogas.

Nombre:
Ingrese el nombre del nuevo proyecto.

Descripción:
Digite una breve descripción del proyecto que está registrando

Fecha de inicio:
Seleccione la fecha en la que dará inicio el proyecto, mediante un pequeño calendario que aparece al momento de dar clic en el campo.

Fecha de finalización:
Seleccione la fecha en la finalizara el proyecto, el campo aparecerá deshabilitado mientras no se haya seleccionado una fecha de inicio, posteriormente se presentara un pequeño calendario al momento de dar clic en el campo.

Responsable:
Seleccione el responsable del proyecto. Para seleccionar un responsable tiene que dar clic en el campo para realizar dicha acción.

Guardar:
Almacena a la base de datos la información ingresada en el formulario.

(*) Campo requerido:
Al aparecer este símbolo en alguna etiqueta quiere dar a entender que el campo es requerido u obligatorio. En otras palabras si no es ingresado el dato solicitado en el campo, no se podrá almacenar en la base de datos del sistema.
