Permite realizar diferentes acciones en los usuarios como registrar, visualizar y modificar, así como tambien podrá generar reportes de manera general.

1. Nuevo:
Registra un nuevo usuario.

2. Visualizar:
Muestra el contenido del usuario.

3. Modificar:
Actualiza los datos ingresados anteriormente del usuario seleccionado.

4. Dar de Baja:
Cambiar el estado del usuario activo a inactivo.

5. Usuarios Inactivos:
Muestra el listado de usuarios inactivos, ademas permite reestablecer el estado del usuario a activo.

Nombre:
Ingrese el nombre de la persona que utilizará el usuario.

Apellido:
Ingrese el apellido de la persona que utilizará el usuario.

Email:
Ingrese el correo del usuario.

Nombre de usuario:
Ingrese el nombre que le asignará al usuario.

Contraseña:
Ingrese la contraseña que le asignará al usuario, esta deberá llevar signos, numeros y letras para una mayor seguridad.

Repita contraseña:
Ingrese nuevamente la contraseña.

DUI:
Ingrese el DUI solamente si la persona es mayor de 18 años.

Tipo:
Seleccione el tipo de usuario. En el sistema existen dos tipos de usuarios: Administrador, permite acceder a todo el sistema completo; Digitador, permite acceder solamente al registro de ejemplares y localizaciones.

Guardar:
Almacena a la base de datos la información ingresada en el formulario.

(X)Cerrar:
Cierra el formulario sin almacenar los datos ingresados.

(*) Campo requerido:
Al aparecer este símbolo en alguna etiqueta quiere dar a entender que el campo es requerido u obligatorio. En otras palabras si no es ingresado el dato solicitado en el campo, no se podrá almacenar en la base de datos del sistema.

Pasos para visualizar el contenido del usuario.

Paso 1:
Seleccióne el usuario que desea visualizar.

Paso 2:
Se habilitara el botón Visualizar para luego hacer clic y ver la información del usuario seleccionado.

Paso 3:
Si desea cerrar el formulario, haga clic en el botón X.

Pasos para modificar el usuario.

Paso 1:
Seleccióne el usuario que desea modificar.

Paso 2:
Se habilitara el botón Modificar, luego hacer clic y el formulario mostrara los datos del usuario para que puedan ser editados.

Paso 3:
Realize los cambios que desea y haga clic en el botón Guardar.

Paso 4:
Si desea cerrar el formulario, haga clic en el botón X.

Pasos para dar de baja al usuario.

Paso 1:
Seleccióne el usuario que desea dar de baja.

Paso 2:
Se habilitara el botón Dar de Baja, luego hacer clic y el formulario mostrara un mensaje.

Paso 3:
Hacer clic en el botón Si, caso contrario hacer clic en el botón No.

Paso 4:
Si desea cerrar el formulario, haga clic en el botón X.

Paso para administrar los usuarios inactivos.

Paso 1:
Hacer clic en el botón Usuarios Inactivos, para dirigirse a los usuarios que se encuentran inactivos.
